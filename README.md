# Files
## Data
ZIP folder containing images and binary masks for the Toshiba Capassee and Siemens Elegra scanners

## addContrast.m
MATLAB helper function to create hypoechoic structures using the additive method

## noise2pipes_batch.m
MATLAB code to batch process images using either the additive or subtractive approach
